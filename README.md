# Guess the Number Game (C)

A simple console-based Guess the Number game written in C.

## Description

The program generates a random number, and the player tries to guess it. After each guess, the program provides hints indicating whether the guess is too high or too low until the correct number is found.

## Features

- Random number generation
- User input handling
- Hint system (higher/lower)
- Tracks the number of attempts

## How to Compile

Using GCC:

```bash
gcc main.c -o guess
